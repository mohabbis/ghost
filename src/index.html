Welcome to Ghost

Ghost watches the repetitive things you do on screen, then replays them for you — record a task once, run it forever.

🔒 Ghost never records passwords or secure text fields.

🔑

Grant recording permissions

To watch and replay clicks and keystrokes, macOS needs you to allow Ghost under System Settings → Privacy & Security in both Accessibility and Input Monitoring.

You're all set

Three steps to your first automation:

  1. Start Recording, then do your task once.
  2. Hit Stop when you're done.
  3. Press Replay — Ghost repeats it for you.
